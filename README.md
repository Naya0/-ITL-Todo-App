# ITL Todo App

## Особенности

- Добавление новых задач с проверкой на дубликаты и пустые значения.
- Редактирование задач с мгновенным сохранением изменений.
- Удаление задач.
- Изменение статуса задачи (текущая / завершенная) с поддержкой фильтрации.
- Фильтрация задач по статусу: все, текущие, завершенные.
- Переключение между двумя видами отображения: с фильтром и обычным списком.
- Локальное хранение данных с помощью `localStorage`.
- Контекстное управление состоянием задач для удобного доступа из любого компонента.



## Клонироивание репозитория

```bash
git clone https://github.com/Naya0/-ITL-Todo-App.git
cd itl
```

## Установка зависимостей
npm install

## Запуск в режиме разработки:
npm run dev
